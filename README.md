# typescript-ilogger

## Install
```
npm install typescript-ilogger@latest
```

## Logger
The ILogger has 6 different log levels that are displayed in hierarchial order on which logs are actually logged:
- Debug
- Trace
- Error
- Warning
- Information
- Off

Example Log:
```javascript
[KMSHelper.DecryptAsync]-Inputs: { encryptedValue: 'asdf' }
```
